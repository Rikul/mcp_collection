# Outline Parser MCP Server

## What it does

Converts indented bullet outlines to/from a JSON tree.

## When to use it

- Turn brainstorming outlines into structured JSON for programmatic manipulation.
- Render structured plans back into a readable outline.

## Tools

- `outline_to_json(outline: str)`
- `json_to_outline(json_text: str)`

## How to run

```bash
uv run outline-parser-mcp
```
