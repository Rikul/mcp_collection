# Countries & Currencies MCP Server

## What it does

Looks up country information (REST Countries) and provides exchange rates + currency conversion.

## When to use it

- Country metadata (capital, region, population, languages, currencies).
- Quick FX rate lookup or converting an amount.

## Tools

- `get_country_info`
- `search_countries_by_region`
- `search_countries_by_currency`
- `search_countries_by_language`
- `get_exchange_rates`
- `convert_currency`
- `compare_countries`

## How to run

```bash
uv run countries-currencies-mcp
```
