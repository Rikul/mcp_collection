# uuid-tools-mcp

A tiny MCP server that provides UUID helpers:

- Generate UUIDv4
- Generate UUIDv5 from a namespace + name
- Validate UUID strings
- Inspect UUID details (version, variant)

This server **does not** make any network calls.

## Install

```bash
pip install -e .
```

## Run

```bash
uuid-tools-mcp
```
