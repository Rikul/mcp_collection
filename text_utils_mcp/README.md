# Text Utilities MCP Server

A tiny grab-bag of text helpers in the spirit of “100 lines of code” utilities.

## Available tools

- `word_count(text)` – count chars/words/lines
- `sha256(text)` – SHA-256 digest (hex)
- `extract_urls(text)` – extract URLs as plain text (no fetching)
- `normalize_whitespace(text)` – collapse whitespace and trim

## Running

```bash
uv run text-utils-mcp
```
