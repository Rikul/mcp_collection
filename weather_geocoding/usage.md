# Weather & Geocoding MCP Server

## What it does

Geocoding (forward/reverse) via Nominatim and weather/forecasts via Open-Meteo.

## When to use it

- Convert a place name to coordinates, then fetch current weather or a forecast.

## Tools

- `geocode_location`
- `reverse_geocode`
- `get_current_weather`
- `get_weather_forecast`
- `get_weather_by_coordinates`

## How to run

```bash
uv run weather-geocoding-mcp
```
